TP2 - Application Fitness


1.  Le thème est le Fitness, l'application affiche une liste d'exercices avec des images et un compteur de séries interactif.

2. Étapes pour lancer le projet: 
   Ouvrir le projet dans Android Studio.

   Sync Gradle et attendre la fin du chargement.

   Réparer les icônes (si besoin) : Clic droit sur res > New > Image Asset > Finish.

   Lancer l'émulateur et cliquer sur le bouton Run (Flèche verte).

3. Emplacement du switch clair/sombre
   Le Switch se trouve en haut à droite de l'écran principal, juste à côté du titre "FitnessApp".
